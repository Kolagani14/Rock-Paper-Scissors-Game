# 🪨 Rock Paper Scissors Game

A command-line Rock Paper Scissors game built using Python where a user plays against the computer. The game uses random selection for the computer’s move and includes a scoring system to track performance across multiple rounds.

## 🚀 Features
- Play against computer with random choices
- Supports Rock, Paper, and Scissors inputs
- Score tracking system (User vs Computer)
- Continuous gameplay with menu options
- Input validation for correct choices

## 🛠️ Technologies Used
- Python
- Random module

## ▶️ How It Works
1. User selects the play option
2. Enters a choice (rock, paper, or scissors)
3. Computer randomly selects its move
4. Winner is decided based on game rules
5. Score is updated and displayed after each round
